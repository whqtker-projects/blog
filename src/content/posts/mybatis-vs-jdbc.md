## JDBC의 작업 방식

- 연결 획득, SQL 실행, 결과 매핑까지 직접 처리하는 흐름을 정리한다.

## MyBatis가 줄여 주는 반복 작업

- 파라미터 바인딩, 결과 매핑, 자원 처리 측면에서 달라지는 점을 본다.

## 장단점 비교

- SQL 제어력, 생산성, 추상화 수준 관점에서 비교한다.

## 어떤 상황에서 적합한가

- CRUD 중심 시스템과 복잡한 조회 요구사항에서의 선택 기준을 정리한다.
